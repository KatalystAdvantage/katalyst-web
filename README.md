# Katalyst Web

1-page longscroll Katalyst Website

Code style
- SMACSS used for CSS structure

Tech used
- Node 6.7.0
- ASP .NET MVC

Installation
- Frontend:
  - Use Node 6.7.0
  - run npm install on root folder

How to use?
- run gulp watch to compile css
- open .sln file in VS Express for Web 2015 to build and run project
